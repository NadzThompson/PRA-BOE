================================================================================
NOVA METADATA FIELDS
================================================================================

Generated for each publication:

IDENTIFIERS:
  - doc_id: boe.ps.2-24.2024 (unique, includes year)
  - doc_family_id: boe.ps.2-24 (version-independent)

DOCUMENT INFORMATION:
  - title: Full publication title
  - short_title: Abbreviated form
  - document_class: policy_statement|supervisory_statement|consultation_paper|letter
  - guideline_number: PS2/24, SS1/14, etc.

REGULATORY CONTEXT:
  - regulator: "BoE"
  - jurisdiction: "United Kingdom"
  - authority_class: primary_normative|interpretive|context
  - authority_level: 1-3 (1=highest)
  - nova_tier: 1-3 (1=highest)

DOCUMENT PROPERTIES:
  - source_type: "external_regulatory"
  - status: "active"
  - effective_date_start: ISO date (2024-12-01)
  - sector: ["Banking"] or ["Insurance"] or ["All"]

CONTENT FEATURES:
  - contains_definition: Boolean
  - contains_formula: Boolean
  - contains_requirement: Boolean

METADATA:
  - source_url: Original publication URL
  - published_date: ISO date
  - pdf_links: Array of PDF links
  - related_links: Array of related publications

================================================================================
TROUBLESHOOTING
================================================================================

Issue: No URLs discovered
  Cause: BoE pages use JavaScript pagination
  Solution: Use seed file mode instead

Issue: Module not found
  Solution: pip install requests beautifulsoup4 html2text

Issue: Permission denied
  Solution: chmod 755 output_dir or use writable location

Issue: Request timeout
  Note: Scraper retries automatically (3 attempts)

Issue: Empty content extracted
  Check: Verify BoE page structure hasn't changed
